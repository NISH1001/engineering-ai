# Simulated Annealing
    Let s = s0
    For k = 0 through kmax (exclusive):
        T ← temperature( kmax/(k+1) )
        Pick a random neighbour, snew ← neighbour(s)
        If P(E(s), E(snew), T) ≥ random(0, 1):
            s ← snew
    Output: the final state s
